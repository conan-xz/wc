# 微信小程序 WebSocket 调试工具

这是一个用于调试微信云开发容器服务 WebSocket 的小程序。

## 功能特性

- 连接云开发容器服务的 WebSocket
- 实时收发消息
- 显示连接状态和消息历史
- 清空消息功能

## 使用说明

1. **导入项目**: 使用微信开发者工具导入本项目

2. **配置云开发**:
   - 确保已在微信公众平台开通云开发
   - 环境ID: `prod-5gg03znv016787f1`
   - 服务名称: `express-v2qc`

3. **编译运行**: 点击开发者工具的编译按钮

4. **连接服务**:
   - 点击"连接云服务"按钮
   - 连接成功后会自动发送一条空消息

5. **发送消息**:
   - 在输入框输入内容
   - 点击"发送"按钮或回车

## 注意事项

- 需要使用支持云开发的微信基础库版本（建议 2.12.0 以上）
- 确保云开发环境和容器服务已正确配置并运行
- WebSocket 路径为 `/ws`
